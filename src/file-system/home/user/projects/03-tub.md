



## *Tub*
## 2026

### • TypeScript, Node.js
### • React, Video Streaming
Tub is an open-source YouTube clone designed to showcase modern web technologies for video streaming and user interaction. The project allows users to upload, watch, and comment on videos, as well as like and subscribe to channels. Built with React for the frontend and Node.js for the backend, Tub delivers a responsive, scalable platform for video sharing and community engagement.
